# 디지털 바다 유리 (Digital Sea Glass) 프로젝트 개발 요약

본 문서는 지금까지 진행된 '디지털 바다 유리' 애플리케이션의 개발 사항 및 주요 기능, 디자인 결정을 정리한 보고서입니다.

---

## 1. 프로젝트 개요
'디지털 바다 유리'는 사용자가 자신의 생각이나 감정을 기록하고, 이를 바다에 떠다니는 유리 조각처럼 시각화하여 마음을 치유하고 정리할 수 있도록 돕는 힐링 인터랙션 앱입니다.

---

## 2. 주요 구현 기능

### 🔐 사용자 인증 시스템 (Authentication)
*   **ID 기반 인증**: 복잡한 이메일 형식 대신 사용자가 정의한 간단한 '아이디'로 가입 및 로그인 가능. (내부적으로는 `username@seaglass.internal` 가상 이메일로 처리되어 Supabase와 호환)
*   **보안**: 비밀번호 기반의 안전한 보안 시스템 및 로그아웃 기능 구현.
*   **사용자 경험**: 가입 즉시 로그인 상태가 유지되도록 이메일 확인 절차 최적화.

### 🌊 인터랙션 및 물리 엔진 (Physics)
*   **Matter.js 기반 물리 환경**: 중력이 없는 공간에서 유리 조각들이 유영하며 벽에 부딪히면 튕겨 나가는 입체적인 움직임 구현.
*   **파동 인터랙션 (Ripple Effect)**: 화면을 클릭하거나 터치하면 시각적인 파동이 발생하며, 주변의 유리 조각들이 물리적으로 밀려나는 생동감 있는 효과 추가.
*   **움직임 제어**: 조각들이 화면 밖으로 나가지 않도록 고정된 프레임(Wall) 설치 및 유기적인 '자율 영동' 로직 적용.

### 💠 데이터 및 카테고리 관리
*   **유리 조각 기록**: 사용자의 메시지를 유리 조각 형태로 생성, 조회, 수정, 삭제(CRUD) 기능.
*   **동적 카테고리**: '잡생각', '추억', '기분' 등 카테고리를 자유롭게 추가/삭제할 수 있으며, 각 카테고리에 맞는 고유한 색상 부여.
*   **실시간 연동**: 모든 데이터는 Supabase 데이터베이스에 실시간으로 저장 및 동기화.

---

## 3. UI/UX 디자인 시스템

### 🎨 디자인 테마: 미니멀리즘 & 글래스모피즘
*   **배경 색상**: 여러 차례의 조정을 거쳐 '맑고 깨끗한 블루(#3182ce)'를 최종 적용. 시각적 피로를 줄이고 맑은 바다의 느낌을 강조.
*   **글래스모피즘 (Glassmorphism)**: 유리 조각과 UI 요소에 투명도와 블러 효과를 적용하여 몽환적이고 고급스러운 질감 표현.
*   **프레임 디자인**: 조각들이 노는 구역을 시각적으로 구분해주는 반투명한 원형 프레임 적용.

---

## 4. 기술 스택 (Tech Stack)
*   **Frontend**: Next.js (App Router), React, TypeScript
*   **Styling**: Tailwind CSS, Vanilla CSS (globals.css), Framer Motion (애니메이션)
*   **Physics Engine**: Matter.js
*   **Backend/DB**: Supabase (Auth, Database)

---

## 5. 변경 이력 요약
1.  **초기 구축**: 물리 엔진 및 기본 메시지 저장 기능 구현.
2.  **보안 강화**: 이메일 주소 방식에서 ID 기반 로그인 방식으로 가다듬음.
3.  **UI 고도화**: 글래스모피즘 스타일 적용 및 카테고리별 동적 컬러 시스템 구축.
4.  **인터택션 최적화**: 클릭 시 파동 효과 및 물리적 반동 효과 정교화.
5.  **미니멀리즘 정착**: 복잡한 배경 효과(빛줄기, 비네팅 등)를 제거하고 사용자의 기록에만 집중할 수 있는 깔끔한 단색 배경으로 최종 조정.

---

## 6. 향후 발전 방향
*   **감성 효과 추가**: 조각 삭제 시 파도에 씻겨 나가는 듯한 애니메이션 개발.
*   **사운드 디자인**: 물소리, 유리의 맑은 충돌음 등 힐링 사운드(BGM/SFX) 추가.
*   **데이터 시각화**: 내가 남긴 감정 조각들의 비율을 보여주는 통계 페이지.
