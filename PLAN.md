# [기획서] 디지털 바다 유리 (Digital Sea Glass)

## 1. 프로젝트 개요
'디지털 바다 유리'는 사용자의 행복한 메시지를 시각화한 힐링 애플리케이션입니다. 바다에 버려진 유리 조각이 파도에 깎여 둥글게 변하듯, 시간의 흐름에 따라 변화하는 메시지 조각들을 보며 정서적 안정을 얻는 것을 목표로 합니다.

## 2. 핵심 컨셉
- **무중력 공간 (Zero Gravity)**: 화면은 물속 혹은 우주와 같은 무중력 환경을 시뮬레이션합니다.
- **유리 조각 (Sea Glass)**: 사용자가 입력한 메시지는 각기 다른 색상과 형태의 유리 조각으로 변환됩니다.
- **시간의 흔적 (Aging Process)**: 최신 데이터는 각진 형태를 유지하지만, 오래된 데이터일수록 파도에 깎인 듯 모서리가 부드럽고 둥글게 표현됩니다.

## 3. 주요 기능
### A. 물리 환경 구현
- **중력 0 설정**: 물체들이 특정 방향으로 떨어지지 않고 자유롭게 유영합니다.
- **느린 유영 (Damping & Speed Limit)**: 모든 오브젝트는 일정한 속도 이하로 천천히 움직여 시각적 편안함을 제공합니다.
- **완전 탄성 충돌 (Bounciness)**: 벽이나 다른 조각과 부딪힐 때 에너지를 거의 잃지 않고 튕겨 나갑니다.

### B. 데이터 및 비주얼
- **메시지 입력**: 사용자가 짧은 행복 문구를 입력하면 화면에 즉시 유리 조각 오브젝트로 나타납니다. (초기 단계에서는 로컬 상태로 관리)
- **동적 렌더링**: 입력된 데이터를 기반으로 유리 조각 오브젝트를 생성합니다.
- **적응형 디자인**: 
  - 색상: 파스텔톤 랜덤 지정.
  - 형태: 작성일(Timestamp)로부터 현재까지의 시간에 따라 `border-radius` 또는 렌더링 형태를 둥글게 조정합니다.

### C. 인터랙션
- **클릭 및 터치**: 조각을 선택하면 부드러운 애니메이션과 함께 메시지 텍스트가 나타납니다.
- **마우스 반응**: 마우스 커서 주변으로 조각들이 살짝 밀려나거나 끌려오는 부드러운 효과를 추가합니다.

## 4. 기술 스택
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Vanilla CSS
- **Physics Engine**: Matter.js

## 5. 단계별 개발 계획
1. **1단계**: Next.js 환경 구축 및 Matter.js 물리 엔진 기본 세팅 (무중력 구현).
2. **2단계**: 유리 조각 컴포넌트 개발 및 시간 기반 둥글기 변화 로직 구현.
3. **3단계**: 메시지 작성 기능 추가 및 로컬 상태 관리.
4. **4단계**: 시각 효과 고도화 (글래스모피즘, 배경 그라데이션, 부드러운 전이 효과).
